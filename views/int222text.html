Basic html text

HTML has the necessary tools to create all kinds of text, from the very basic to very elaborate. You can use HTML tags to insert hyperlinks into text, to insert images, manipulate font weight, size, color and decoration

These are some of the text properties and the means we can use to specify them:

  • a line break - br
  • bolding - b
  • italicization - i
  • long quotation - blockquote
  • new paragraph - p
  • subscripting - sub
  • superscripting - sup
  • underlining - u

Text with CSS - Example #1

"When one door closes, another opens - but often we look so long and so regretfully upon the closed door that we fail to see the one that has opened for us".

In this example: the word "when" is enclosed in b tag for bolding, the word "opens" is used with span, and background-color with value fff000 for yellow.
The CSS properties used are: "color" (#000099), "direction" (rtl), "letter-spacing" (2pt), "background-color" (#fff000), and "font-family" (fantasy).

Text with CSS - Example #2

"There are two things to aim at in life;
first to get what you want, and after that to enjoy it.
Only the wisest of mankind has achieved the second".

In this example, the words "first" and "after that" are bolded using span with properties: "letter-spacing" (0.2em), "color" (#ff0000), and "text-decoration" (underline).
The CSS properties applied are "line-height" (100%), "text-align" (center), "text-decoration" (blink), and "font-size" (large).

Text with CSS - Example #3

"Many do with opportunities as children do at the seashore; they fill their little hands with sand, and then let the grains fall through, one by one".

The word "children" is modified using span with properties "color" (#009900), and "text-decoration" (overline).
The CSS properties implemented are "text-indent" (10pt), "text-transform" (uppercase), "vertical-align" (top), and "font-weight" (bold).

Text with CSS - Example #4

"All compromise is based on give and take, but there can be no give and take on fundamentals. Any compromise on mere fundamentals is surrender - for it is all give and no take".

The CSS properties I have included here are "word-spacing" (1em), "text-justify" (distribute), "text-shadow" (2px 2px #aaaaaa), "font-style" (italic), and "font-variant" (small-caps).
