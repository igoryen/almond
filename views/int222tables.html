Basic html table

We can use HTML tags to create tables. HTML has tags for every part of a table: the column, the row and the cell.

Table tags are used between two table tags (table). Below is the list of the basic HTML table tags and their purpose:


  • "table" - table
  • "caption" - table caption
  • "tr" - table row
  • "th" - table heading
  • "td" - table details (or cells)
